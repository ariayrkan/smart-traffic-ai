:

🔝 Top Navigation Bar — Lets you switch between the 4 simulation scenarios (Normal, Spoofing Attack, Heavy Traffic, Recovery). The clock shows real time. The green "Live" pill confirms the simulation is running.

🟠 Red Alert Bar — Only appears during Spoofing. It's a security notification that tells your viewer the system detected an attack and activated fallback mode. Great for demos.

📊 Status Strip (6 small cards) — A quick-glance summary of system health at that moment. Step is the simulation counter. Mode shows what logic is running (AI or fallback). Risk and Sensor Trust change color based on threat level. Failover shows whether the backup rule-based system is active. AI Confidence shows how certain the model is.

🛣️ Intersection View — The live top-down map of the junction. Green vehicles are traveling N/S, blue are E/W. Traffic lights in the 4 corners switch phases automatically. Vehicles stop at red lights and fade slightly when waiting. The footer shows Real Count vs Sensor Count — if those two numbers differ a lot, that's the anomaly the system detects.

📋 Event Log — A timestamped log of everything happening: INFO messages during normal ops, WARN during heavy traffic, ERROR during attacks, OK when things recover. Newest events appear at the top.

⚙️ Controls — Playback speed changes how fast the simulation ticks. Pause stops everything. Auto Demo cycles through all 4 scenarios automatically every 9 seconds — useful for presentations. Simulate 
Emergency randomly picks Ambulance/Fire/Police and triggers the override for 7 seconds.

📈 Metrics Row (4 cards) — Real Count vs Sensor Count lets you spot spoofing instantly. Avg Speed drops in heavy traffic. Wait Time spikes when queues build up. The colored badges (red = problem, green = good) make it easy to see at a glance.

📉 Live Analytics Chart — Plots both real and sensor vehicle counts over the last 60 seconds. In normal mode the two lines are almost identical. In spoofing mode the blue sensor line spikes way above the green real line — that gap is the attack.

 AI Decision Engine — Shows what the AI is currently deciding (e.g. EXTEND_GREEN or SAFE_MODE), how confident it is, and whether each security check is passing. In spoofing mode, Sensor Validation and Data Integrity flip to red fail states.

🚨 Emergency Override — Shows whether an emergency vehicle was detected. When triggered, it pulses red, names the vehicle type, and forces all lights green on the emergency route for 7 seconds.

🛡️ Security Threat Model — Static reference panel at the bottom. Maps your 4 threat types (from your project plan) to NIST AI RMF controls and MIT AI Risk Repository categories. This is your academic/security framework justification, shown visually.
